How to Install

1

Enable sideloading on your TV

Go to Settings → System → About → click Build number 7 times to unlock developer mode. Then enable Unknown Sources in Developer Options or Security settings.

2

Install the Downloader app

Search for Downloader by AFTVnews on the Google Play Store on your TV and install it. It's free.

3

Enter the download URL in Downloader

Open Downloader, tap the URL bar, and type:

https://github.com/ctsw05/nexustv-android/releases/latest/download/nexustv.apk
4

Install and enjoy

Tap Install when the download finishes. Open Nexus TV from your apps list, go to Settings, and add your M3U playlist URL.
