# 📺 Nexus TV

A free, open-source IPTV player for Android TV and Google TV. Supports M3U playlists, XMLTV TV guides, favorites, and automatic updates.

<br>

## ⬇️ Download & Install

**Latest release:** [github.com/ctsw05/nexustv-android/releases/latest](https://github.com/ctsw05/nexustv-android/releases/latest)

👉 **[Full install guide for Google TV / Android TV](INSTALL.md)**

> Install in about 2 minutes using the free **Downloader** app. No Play Store needed.

<br>

## ✨ Features

- 📋 **M3U playlist support** — add any IPTV playlist by URL or local file
- 📺 **TV Guide (EPG)** — XMLTV support with current & upcoming programme info
- ⭐ **Favorites** — star your most-watched channels for quick access
- 🕐 **Recent** — picks up where you left off
- 🔇 **Hide channels** — remove channels you never watch
- 📶 **Smart buffering** — adaptive buffer that adjusts to your connection
- 🔄 **Auto-retry** — silently reconnects when a stream drops
- 🆕 **Auto-updates** — notifies you when a new version is available
- 🎮 **Remote-friendly** — fully D-pad navigable, no mouse needed
- 📱 **Multiple playlists** — save and switch between different providers

<br>

## 🚀 Quick Start

1. Install the app ([see install guide](INSTALL.md))
2. Open Nexus TV and press **Settings**
3. Enter your playlist name and M3U URL from your IPTV provider
4. Press **Save** — channels load automatically
5. Optionally add your EPG URL for TV guide data

<br>

## 🎮 Remote Control

| Key | Action |
|-----|--------|
| Up / Down | Scroll channel list |
| Left | Jump to top navigation bar |
| OK / Select | Play channel (enters fullscreen) |
| Long-press OK | Add to Favorites / Hide channel |
| CH ▲ / CH ▼ | Previous / Next channel (fullscreen) |
| Back | Exit fullscreen / back to list |
| Back × 2 | Exit app |
| Info key | Show help screen |
